How to run - 
* put this folder in the triton\exampleremoras folder for the version of triton you're using (this remora is designed to work with the 1.92.20160524  version of triton)
* run triton, go to Remoras->Add Remora, and select this folder
* restart triton
* Remoras->Blue Whale B Detector
* Select the .m file with the threshold/kernels/directories you want to use (see example parameters file for format)
* Press continue and run!
