Triton User Guide

 

Triton software package is used to display XWAV and WAV data from HARPs, ARPs, and hydrophone arrays as timeseries, spectra, spectrograms and LTSAs (long-term spectral averages) plots.

 

Various tools are included to convert raw formats to XWAV, decimate broad-band data, and generate LTSAs.
